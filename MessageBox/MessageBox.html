JavaScript:

  • MessageBox.show(message)
  • MessageBox.show(message, title)
  • MessageBox.show(message, title, buttonActions)
  • MessageBox.show(message, title, buttonsList)
  • MessageBox.show(message, title, buttonsList, defaultButtonIndex)

Description:

MessageBox is an extenstion of javascript that provide facility to display html message as popup with custom action button(s). It is useful if we want to display html as message or multiple action buttons and want to take decision accordingly. It is an replacement of javascript:alert box. Where alert box have only one button 'Ok' and display only text message.

Features of MessageBox

  • can display html message
  • can have multiple user defined buttons
  • can take decision on button click
  • can apply css styling
  • can not stop the execution of rest of program
  • can take decision on close
  • can define promise like structure
  • can be used many ways
  • have multiple overloaded structure

Syntax :

MessageBox.show( message, [title], [buttonsActionList], defaultButtonIndex ).buttonActionMethods(fn)

  • message : Required [text/html/object] that display as message in MessageBox.
  • title : [Optional] title of MessageBox popup window. if not passed then default title is "MessageBox"
  • [buttonsActionList] : [Optional] Array of Button Names or
    Array of Objects
    {
    btnCaption | btnText | caption | text,
    buttonAction | btnAction | action | fn,
    isActive | isDefault | active | default,
    cssClass | class | className
    }

    if not passed default "OK" button will display.
  • defaultButtonIndex : [Optional] zero base index value of which button will be focused when MessageBox have multiple button
  • .buttonActionMethods(fn) : define callBack methods that will execute after different buttons of MessageBox clicked.
    • .then(fn) : callBack when "OK" button will clicked.
    • .onClose(fn) : callBack when MessageBox close button will clicked.
    • .obButtonn(fn) : callBack when MessageBox buttons will clicked. As 'onButton1', 'onButton2', 'onButton...'

Examples :

MessageBox.show("Hello World");
